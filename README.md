# Tema 3 – Desenvolvendo a Lógica do Jogo

Este trabalho tem como objetivo demonstrar a lógica de um jogo
utilizando estruturas de decisão e repetição.

O jogo consiste em uma adivinhação onde o usuário deve descobrir
um número secreto, recebendo dicas se o número é maior ou menor,
até acertar.

Conceitos utilizados:
- Estruturas condicionais (if, elif, else)
- Estrutura de repetição (while)
- Controle de fluxo do jogo

Linguagem utilizada: Python
